# nextnanopy-community-examples

This repository contains **community examples** using [`nextnanopy`](https://github.com/nextnano-GmbH/nextnanopy).

The examples are **not officially maintained** and may depend on specific versions of nextnano solvers  and their output formats (paths, keywords, file structures).

As nextnanopy and the underlying solvers evolve, some examples may **stop working**.
They are provided as **reference material and starting points**, without compatibility guarantees.

Each example typically documents the versions it was tested with.

Some examples were used to gerenerate data or plots for the tutorial in nextnano documentation:

- cylindrical-well: https://www.nextnano.com/docu/nextnanoplus/latest/tutorials/2D_quantum_corral.html

- ellipsoidal-cdse-quantum-dot: https://www.nextnano.com/docu/nextnanoplus/tutorials/QD-CdSe-ellipsoidal_zb_II-IV_Ferreira_BJP_2006_3D.html

- infinite-quantum-well-sweep: https://www.nextnano.com/docu/nextnanoevo/tutorials/scipy/1D_infinite_well_optimization.html

- infinite-well-exciton: https://www.nextnano.com/docu/nextnanoplus/latest/tutorials/SiPho-G/1D_exciton_in_infinite_quantum_well.html

- interband-tunneling: https://www.nextnano.com/docu/nextnanoplus/latest/tutorials/1D_interband_tunneling_in_nitride_junction.html

- ir-mqw-led: https://www.nextnano.com/docu/nextnanoplus/latest/tutorials/1D_InGaAs_MQW_LED.html

- UVC_LED_wz: https://www.nextnano.com/docu/nextnanoplus/tutorials/UVC-LED_wz_III-N_1D.html
